BXXC
=====

[![Built with Grunt](https://cdn.gruntjs.com/builtwith.png)](http://gruntjs.com/)

> Node.js frontend for BXXC based on [Express](http://expressjs.com/).

### Setup

```shell
$ npm install
```

### Development

```shell
$ grunt dev
```

### Sync MongoHQ

```shell
$ grunt mongohq
```
